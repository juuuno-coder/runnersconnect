- [ ] "상세보기" 및 "📊 기록 통계" 링크 동작

#### 2.2 필터링 및 검색
1. 종목 필터: "벚꽃 10K 코스" 선택
2. 성별 필터: "남성" 선택
3. 검색창에 참가자 이름 입력

**체크리스트**:
- [ ] 종목 필터 동작
- [ ] 성별 필터 동작
- [ ] 연령대 필터 동작
- [ ] 이름/이메일 검색 동작
- [ ] 필터 조합 가능
- [ ] "초기화" 버튼으로 필터 해제

#### 2.3 CSV 다운로드
1. "CSV 다운로드" 버튼 클릭
2. 다운로드된 파일 확인

**체크리스트**:
- [ ] CSV 파일 다운로드 성공
- [ ] 파일명 형식: `2026_여의도_벚꽃_러닝_페스티벌_participants_[타임스탬프].csv`
- [ ] CSV 내용: 등번호, 이름, 성별, 연령대, 종목, 전화번호, 이메일, 주소, 신청일

### **Phase 6: 참가 기록 관리 (45분)**

#### 6.1 기록 목록 확인
1. 대시보드 → "기록 관리" 클릭
2. 현재 기록 확인

**체크리스트**:
- [ ] 완주자 수: **25명**
- [ ] 완주율: **50%** (25/50)
- [ ] 순위, 등번호, 이름, 종목, 완주 시간, 페이스 표시

#### 6.2 기록 일괄 업로드 (CSV)
1. "📤 기록 업로드" 버튼 클릭
2. "샘플 CSV 다운로드" 링크 클릭
3. 샘플 CSV 파일 확인
4. 테스트 CSV 파일 작성:
   ```csv
   등번호,완주 시간
   26,00:45:30
   27,00:48:15
   28,00:52:00
   29,00:43:20
   30,00:50:45
   ```
5. "파일 선택" → 작성한 CSV 업로드
6. "업로드" 버튼 클릭

**체크리스트**:
- [ ] 샘플 CSV 다운로드 동작
- [ ] CSV 파일 업로드 성공
- [ ] 성공 메시지: "5건의 기록이 업로드되었습니다."
- [ ] 기록 목록에 새 기록 추가됨
- [ ] 완주율 업데이트: **60%** (30/50)

#### 6.3 기록 업로드 에러 처리
1. 잘못된 CSV 파일 작성:
   ```csv
   등번호,완주 시간
   999,00:45:30
   26,invalid_time
   ```
2. 업로드 시도

**체크리스트**:
- [ ] 에러 메시지 표시
- [ ] "등번호 999: 참가자를 찾을 수 없거나 시간 형식 오류"
- [ ] "등번호 26: 시간 형식 오류" (이미 중복 기록)
- [ ] 에러 건수 표시

- [ ] "벚꽃러닝 커뮤니티의 다른 대회" 섹션 표시 (없을 수 있음)
- [ ] "비슷한 시기의 다른 대회" 섹션 표시
  - 서울 국제 마라톤
  - 부산 바다 마라톤
- [ ] 대회 카드 클릭 시 해당 대회 페이지로 이동

#### 10.3 굿즈 쇼핑
1. 대회 상세 페이지 → "대회 굿즈 스토어 🛍️" 클릭
2. 상품 목록 확인
3. 상품 클릭 → 상세 페이지
4. "장바구니 담기" 클릭
5. 장바구니 확인
6. "주문하기" 클릭
7. 배송지 입력 → 주문 완료

**체크리스트**:
- [ ] 상품 목록 표시 (활성 상태, 재고 있는 것만)
- [ ] 상품 이미지, 이름, 가격 표시
- [ ] 장바구니 담기 동작
- [ ] 장바구니 수량 조절
- [ ] 주문 생성 성공
- [ ] 주문 내역 페이지로 리다이렉트
