# Runner's Connect OKR (2026 Q1)

## 프로젝트 개요
**현재 작업 중인 코드/기능**: B2B 대회 운영 관리 플랫폼 (주최자 백오피스, 정산 시스템, 굿즈 판매)
**최종 목적**: 축제/행사 기획사와 커뮤니티 주최자들이 러닝 대회를 쉽고 효율적으로 운영할 수 있도록 지원
**핵심 차별화**: 포스터 OCR 자동 입력, 맞춤형 홈페이지 제작, 투명한 정산 시스템

---

## 🎯 Objective (목표)

**"축제 기획사들이 Runner's Connect 없이는 대회를 못 열겠다고 느끼게 만들기"**

### 목표 해설
단순히 기능을 구현하는 것이 아니라, 주최자들이 대회 운영의 모든 과정(홍보 → 신청 → 현장 운영 → 정산)에서 "이 플랫폼이 없었으면 얼마나 힘들었을까"라고 느낄 수 있는 필수 도구로 자리잡는 것이 목표입니다.

---

## 📊 Key Results (핵심 결과)

### KR 1: 주최자 업무 시간 70% 단축
**측정 방법**:
- 수동 등번호 할당 → 자동 할당: 500명 기준 2시간 → 5분
- 참가자 명단 정리 → CSV 자동 다운로드: 1시간 → 1분
- 기록 입력 → 일괄 업로드: 500명 기준 4시간 → 10분
- **목표**: 평균 업무 시간을 기존 대비 70% 이상 단축

### KR 2: 정산 투명성 만족도 95% 이상
**측정 방법**:
- 정산 내역 실시간 확인 가능 (대시보드)
- 수수료 계산 공식 명확 표시 (5% + 500원/명)
- 정산 요청부터 지급까지 소요 시간 평균 3영업일 이내
- **목표**: 주최자 정산 만족도 설문 95% 이상

### KR 3: 플랫폼 핵심 기능 완성도 100% 달성
**측정 방법**:
- Phase 1 (주최자 백오피스): ✅ 100% 완료
- Phase 2 (현장 운영 & 정산): 현재 80% → **100% 목표**
- Phase 3 (크로스 프로모션 & 굿즈): 현재 30% → **80% 목표**
- **목표**: 3개 Phase 평균 완성도 93% 이상

## 🎯 Self-Evaluation (자체 검토)

### ❌ 이것은 OKR이 아닙니다
- "Phase 2 완료하기" → 단순 할 일 목록
- "코드 10,000줄 작성" → Output 중심
- "회의 3번 하기" → 비즈니스 가치 없음

### ✅ 이것이 진짜 OKR입니다
- **Objective**: "주최자들이 필수 도구로 느끼게 만들기" → 비즈니스 가치 중심
- **KR 1**: "업무 시간 70% 단축" → 사용자에게 주는 실질적 가치
- **KR 2**: "정산 만족도 95%" → 신뢰 구축 결과
- **KR 3**: "완성도 93%" → 품질 지표

### 핵심 차이점
우리의 OKR은 "얼마나 많은 코드를 작성했는가?"가 아니라 **"주최자들의 삶을 얼마나 편하게 만들었는가?"**를 측정합니다.
